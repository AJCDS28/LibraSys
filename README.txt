Passos para compilação e execução do programa

1- Abra o programa em sua IDE de escolha, de preferência Eclipse

3- Configure o banco de dados Postgres com o usuario 'postgres' e senha 'udesc', ou altere as configurações de conexão no arquivo Conexão.java para o seu usuario

2- Crie um banco 'libraSys' e execute o arquivo backup do banco no postgres

3- Execute o projeto na IDE
